## 프로젝트 개요
* <b>개발 기간</b> : 2025.09. ~ 2025.11. (2개월) 
* <b>개발 인원</b> : 5명
* <b>핵심 목표</b> : 1. 결재 프로세스 자동화로 업무 효율성 증대 2. 프로젝트 중심의 실시간 협업 환경 구축 3. AI 어시스턴트를 활용한 스마트한 업무 환경 제공
